🧪 Agent Flow Test Guide

📱 Test Case 1: Direct Scan Access

URL: http://localhost:3002/scan
Expected Flow:
  1. Redirects to login page
  2. Agent logs in with: agent@example.com / password123
  3. Goes directly to Scan & Validate page
  4. Can immediately start scanning/manual entry

🏠 Test Case 2: Base URL Access

URL: http://localhost:3002/
Expected Flow:
  1. Redirects to login page
  2. Agent logs in with: agent@example.com / password123
  3. Shows Agent Landing Page with all capabilities
  4. Can access: Passports, Payments, Scan & Validate, etc.

📋 Test Case 3: Navigation from Landing Page

Flow:
  1. Login via base URL (Test Case 2)
  2. Click "Scan & Validate" from Passports menu
  3. Goes to Scan & Validate page

🎯 Expected Results

  • Direct /scan access: Agent lands directly on scan page after login
  • Base URL access: Agent sees full landing page with all options
  • Navigation works: All menu items work correctly
  • Scan functionality: Both camera and manual entry work
